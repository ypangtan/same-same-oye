Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://localhost/yobe-backoffice

Authenticating requests

This API is not authenticated.

Endpoints

Submit a new API request.

Example request:
curl --request POST \
    "http://localhost/yobe-backoffice/api/v1/api-request-operation/create" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"endpoint\": \"\\\"\\/order\\/create\\\"\",
    \"method\": \"\\\"POST\\\"\",
    \"request_body\": {
        \"order_id\": 12345,
        \"amount\": 100
    },
    \"response_body\": {
        \"success\": true,
        \"message\": \"Order created\"
    },
    \"api_name\": \"\\\"Order Creation API\\\"\",
    \"remarks\": \"\\\"Urgent request for new order system.\\\"\"
}"

const url = new URL(
    "http://localhost/yobe-backoffice/api/v1/api-request-operation/create"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "endpoint": "\"\/order\/create\"",
    "method": "\"POST\"",
    "request_body": {
        "order_id": 12345,
        "amount": 100
    },
    "response_body": {
        "success": true,
        "message": "Order created"
    },
    "api_name": "\"Order Creation API\"",
    "remarks": "\"Urgent request for new order system.\""
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (201):


{
    "message": "API request submitted successfully",
    "data": {
        "id": 1,
        "endpoint": "/order/create",
        "method": "POST",
        "status": "pending"
    }
}
Received response:
Request failed with error:

Request   

POST api/v1/api-request-operation/create

Body Parameters

endpoint  string  

The desired API endpoint.

method  string optional  

optional The HTTP method for the API request. Must be one of: GET, POST, PUT, DELETE.

request_body  object optional  

optional The JSON request payload structure.

response_body  object optional  

optional The expected JSON response format.

api_name  string optional  

optional A descriptive name for the API.

remarks  string optional  

optional Additional remarks or comments about the API request.
